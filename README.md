# Library Management System (SQL)

## Overview
A comprehensive relational database schema designed to manage library operations, including book inventory, member registrations, and loan tracking. This project focuses on data integrity and complex relationship management between books and borrowers.

## Key Features
- **Inventory Tracking:** Categorization of books by ISBN, Author, and Title.
- **Member Management:** Systems to track library memberships and registration dates.
- **Loan Logic:** Automated tracking of borrowing dates and return deadlines.
- **Status Monitoring:** Real-time updates on book availability (Available vs. Borrowed).

## Technical Skills Demonstrated
- **Relational Design:** Implementation of Primary Keys and Foreign Keys.
- **Complex Joins:** Advanced SQL queries to link Loans, Members, and Books.
- **Data Validation:** Constraints to ensure unique ISBNs and valid data entries.
- **Business Reporting:** Queries to identify overdue books and active borrowers.

## Business Value
Efficiently automates the manual process of book lending, reducing errors in return dates and improving the overall management of high-volume academic or public libraries.
